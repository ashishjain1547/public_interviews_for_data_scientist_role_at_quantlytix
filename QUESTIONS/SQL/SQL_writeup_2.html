5: What are window functions in SQL? Can you give an example of ROW_NUMBER() or RANK()?

ðŸ”¸ What are Window Functions in SQL?

Window functions perform calculations across a set of table rows that are related to the current row, without collapsing them into a single result (as GROUP BY would). They operate over a "window" of rows defined by the OVER() clause.

ðŸ”¹ Key Characteristics:

  • Do not reduce the number of rows.

  • Let you compute aggregates, ranks, running totals, percentiles, etc.

  • Always use OVER() clause, optionally with PARTITION BY and ORDER BY.

ðŸ”¸ ROW_NUMBER() vs RANK()

FunctionDescription
ROW_NUMBER()Assigns a unique row number to each row within a partition. No ties.
RANK()Assigns a rank to rows within a partition, with gaps for ties.

ðŸ”¹ Example:

Suppose you have a table sales:

regionsalespersonrevenue

EastBob1200
EastCarol1200

ðŸ”¸ Query using ROW_NUMBER() and RANK():

sql
SELECT 
    region,
    salesperson,
    revenue,
    ROW_NUMBER() OVER (PARTITION BY region ORDER BY revenue DESC) AS row_num,
    RANK() OVER (PARTITION BY region ORDER BY revenue DESC) AS rank
FROM sales;
